# E-Commerce Pricing Optimization & Sales Analytics Dashboard (SQL + Excel)

## 📊 Project Overview

This project combines **PostgreSQL** and **Microsoft Excel** to analyze e-commerce sales data, optimize product pricing, and create an interactive business dashboard.

The workflow follows a real-world analytics process:

**Data Collection → SQL Analysis → Business Insights → Excel Dashboard Visualization**

The project helps identify revenue opportunities, pricing improvements, product performance, customer behavior, and sales trends.


## 🛠️ Tools & Technologies

* PostgreSQL
* SQL
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting


## 🗄️ Database Structure

### Products

* Product details
* Categories
* Base prices

### Sales

* Orders
* Revenue
* Discounts
* Quantity sold

### Competitor Prices

* Competitor pricing information

### Reviews

* Customer ratings and feedback


## 🔍 SQL Analysis Performed

* High-profit product identification
* Discount effectiveness analysis
* Competitor price comparison
* Price sensitivity analysis
* Product and category-level pricing recommendations
* Customer rating vs sales analysis
* Monthly and seasonal revenue trends
* Low-performing and high-risk product detection


## 📈 Excel Dashboard Features

* Interactive dashboard with slicers
* Revenue and sales trend analysis
* Product performance tracking
* Customer behavior insights
* Dynamic charts and visualizations
* KPI monitoring for business performance


## 📌 Key Business Metrics

* Total Revenue

* Top Performing Products

* Customer Insights
* Pricing Performance Indicators

This project demonstrates how SQL-driven pricing analysis can be transformed into actionable business insights through an interactive Excel dashboard.
